# 스냅샷 이미지 생성으로 골든 이미지 만들기 및 인스턴스 실행

이 문서는 인스턴스 스냅샷에서 이미지를 생성해 골든 이미지로 활용하고, 새 인스턴스를 실행하는 실습입니다.

## 사전 준비
- Cloud:iA 웹콘솔 접근 가능
- 프로젝트 계정(`project-owner`)으로 로그인 가능
- [인스턴스 스냅샷 롤백하기](./15-rollback-instance-to-snapshot.md)에서 생성한 스냅샷 또는 부팅 가능한 스냅샷 존재

## 개요
1. [스냅샷에서 이미지 생성](#step-1)
2. [이미지 목록 확인](#step-2)
3. [골든 이미지로 인스턴스 생성](#step-3)

<a id="step-1"></a>
## 1단계: 스냅샷에서 이미지 생성
**수행 계정/화면:** `프로젝트 계정` / `프로젝트 페이지`

### 절차
1. `스토리지 > 스냅샷`으로 이동합니다.
2. 대상 스냅샷을 선택하고 `이미지 생성`을 클릭합니다.
3. 스토리지 도메인과 이미지 이름을 입력하고 생성합니다.

### 입력 예시
| 항목 | 값 |
|---|---|
| 스냅샷 | `example-snapshot-golden-001` |
| 스토리지 도메인 | `NFS-SD` |
| 이미지 이름 | `example-golden-image-001` |

### 확인
- 이미지 생성 요청 토스트가 표시됩니다.

<a id="step-2"></a>
## 2단계: 이미지 목록 확인
**수행 계정/화면:** `프로젝트 계정` / `프로젝트 페이지`

### 절차
1. `스토리지 > 이미지`로 이동합니다.
2. 생성한 이미지 이름으로 검색합니다.

### 입력 예시
| 항목 | 값 |
|---|---|
| 검색어 | `example-golden-image-001` |

### 확인
- 목록에 `example-golden-image-001`이 표시됩니다.
- 분류가 `프로젝트 이미지`로 표시됩니다.

<a id="step-3"></a>
## 3단계: 골든 이미지로 인스턴스 생성
**수행 계정/화면:** `프로젝트 계정` / `프로젝트 페이지`

### 절차
1. `가상머신 > 인스턴스`에서 `생성`을 클릭합니다.
2. OS 이미지로 `example-golden-image-001`을 선택합니다.
3. 네트워크/인증 정보를 입력하고 인스턴스를 생성합니다.
4. 인스턴스를 실행합니다.

### 입력 예시
| 항목 | 값 |
|---|---|
| 인스턴스 이름 | `example-instance-from-golden-001` |
| OS 이미지 | `example-golden-image-001` |
| 사용자 이름 | `example-user` |

### 확인
- 인스턴스가 정상 생성되고 `실행 중`으로 전환됩니다.
- 골든 이미지 기준으로 동일 초기 구성이 반영됩니다.

# Cloud:iA 계정 생성하기

이 문서는 Cloud:iA 초기 운영에 필요한 계정 준비 절차를 안내합니다.
프로젝트 멤버 계정을 생성합니다.

## 사전 준비
- Cloud:iA 웹콘솔 접근 가능
- 관리자 계정(`admin`)으로 로그인 가능
- 생성할 멤버 이름/이메일 중복 없음

## 개요
1. [프로젝트 멤버 계정 생성](#step-1)

<a id="step-1"></a>
## 1단계: 프로젝트 멤버 계정 생성
**수행 계정/화면:** `admin 계정` / `어드민 페이지`

### 절차
1. `접근 관리 > 멤버`에서 `생성` 버튼을 클릭합니다.
2. 이름/비밀번호/이메일/관리자 여부를 입력합니다.
3. `생성` 버튼을 클릭합니다.
4. 생성 후 목록에서 방금 생성한 계정을 검색합니다.

### 입력 예시
| 항목 | 값                            |
|---|------------------------------|
| 이름 | `example-member`             |
| 비밀번호 | `initialPassword@123`        |
| 비밀번호 확인 | `initialPassword@123`                |
| 이메일 | `example-member@example.com` |
| 관리자 여부 | `사용 안 함`                     |

### 확인
- 멤버 목록에 `example-member`가 표시됩니다.
- 타입이 관리자 계정이 아닌 프로젝트 멤버로 표시됩니다.
- 에러 팝업/토스트가 발생하지 않습니다.

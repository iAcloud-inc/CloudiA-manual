# 프로젝트 생성하기

이 문서는 프로젝트를 생성하고, 생성한 멤버 계정을 역할에 연결하는 절차를 안내합니다.

## 사전 준비
- [Cloud:iA 계정 생성하기](./01-create-account.md) 완료
- 관리자 계정(`admin`)으로 로그인 가능
- 프로젝트 생성 가능한 리소스 쿼터 확보

## 개요
1. [프로젝트 생성](#step-1)
2. [프로젝트 멤버 역할 연동](#step-2)

<a id="step-1"></a>
## 1단계: 프로젝트 생성
**수행 계정/화면:** `admin 계정` / `어드민 페이지`

### 절차
1. `프로젝트 > 프로젝트 관리`로 이동합니다.
2. `생성` 버튼을 클릭합니다.
3. 프로젝트 기본 정보와 리소스 할당량을 입력합니다.
4. `생성` 버튼을 클릭합니다.

### 입력 예시
| 항목 | 값 |
|---|---|
| 프로젝트 이름 | `example-project` |
| 설명 | `example-project-description` |
| vCPU | `256` |
| 메모리 | `500 GB` |
| 디스크 | `1 TB` |
| 외부 IP | `20` |

### 확인
- 프로젝트 목록에 `example-project`가 표시됩니다.
- 할당량이 `사용량 / 할당량` 형식으로 표시됩니다.

<a id="step-2"></a>
## 2단계: 프로젝트 멤버 역할 연동
**수행 계정/화면:** `admin 계정` / `어드민 페이지`

### 절차
1. `접근 관리 > 멤버`로 이동합니다.
2. `example-member` 상세 화면으로 진입합니다.
3. 하단 `역할` 영역에서 `편집`을 클릭합니다.
4. `역할 추가`를 클릭합니다.
5. 프로젝트/역할을 선택한 후 `저장`합니다.

### 입력 예시
| 항목 | 값 |
|---|---|
| 대상 멤버 | `example-member` |
| 대상 프로젝트 | `example-project` |
| 부여 역할 | `project-owner` |

### 확인
- 저장 성공 토스트가 표시됩니다.
- 역할 목록에 `example-project / project-owner`가 추가됩니다.

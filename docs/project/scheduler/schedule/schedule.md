# 일정

**일정** 기능은 프로젝트의 인스턴스 작업(실행/종료/재부팅/스냅샷)을 지정한 시간 또는 주기로 자동 실행합니다.
일정 유형은 `CRON`과 `RATE`를 지원하며, `기간`과 `일정 완료 후 작업`을 함께 설정할 수 있습니다.

## 일정 목록

### 주요 작업
- `생성`: 새 일정을 생성합니다.
- `편집`: 선택한 1개 일정을 편집합니다.
- `삭제`: 선택한 1개 일정을 삭제합니다.
- `활성화`: 선택한 1개 일정을 활성화합니다. 이미 활성화 상태인 일정은 활성화할 수 없습니다.
- `비활성화`: 선택한 1개 일정을 비활성화합니다. 이미 비활성화 상태인 일정은 비활성화할 수 없습니다.

> 💡 **공통 조건:** 일정 목록은 단일 선택으로 동작하며, `편집/삭제/활성화/비활성화`는 1개 선택 시에만 버튼이 활성화됩니다.

### 검색
- **검색 조건**: `이름`으로 검색할 수 있습니다.

### 테이블 컬럼
| 컬럼 | 설명 |
| --- | --- |
| **이름** | 일정 이름입니다. 클릭 시 상세 화면으로 이동합니다. |
| **ID** | 일정의 고유 식별자입니다. |
| **활성화 상태** | 일정 활성/비활성 상태입니다. |
| **유형** | 일정 유형(`CRON`/`RATE`)입니다. |

| **서비스 이름** | 일정이 실행할 작업 유형입니다. |
| **생성 일시** | 일정이 생성된 날짜와 시간입니다. |

## 일정 생성

`일정 생성` 화면은 **기본 정보**, **일정 패턴**, **기간**, **대상 선택**, **설정** 섹션으로 구성됩니다.

### 기본 정보
- **이름**: 일정 이름을 입력합니다.
- **설명**: 일정 설명을 입력합니다.

### 일정 패턴
- **일정 유형**: `CRON` 또는 `RATE`를 선택합니다.
- **CRON 표현식**: Cron 표현식을 직접 설정합니다.
- **RATE 표현식**: 숫자 + 단위(`minutes`/`hours`/`days`)로 설정합니다.

### 기간
- **시작 일정**
  - `바로 시작`: 시작 시간을 별도로 입력하지 않습니다.
  - `시작 예약`: 시작 날짜/시간을 입력합니다.
- **종료 일정**
  - `종료 없음`: 종료 시간을 설정하지 않습니다.
  - `종료 예약`: 종료 날짜/시간을 입력합니다.

> ⚠️ **주의:** 시작 일정과 종료 일정을 모두 예약한 경우 종료 시각은 시작 시각보다 반드시 늦어야 합니다.

### 대상 선택
- **서비스 선택(필수)**: 아래 4가지 작업 중 하나를 선택합니다.
  - `인스턴스 실행`
  - `인스턴스 종료`
  - `인스턴스 재부팅`
  - `스냅샷 생성`
- **인스턴스 선택**: `추가` 버튼으로 대상 인스턴스를 선택하고, 목록에서 선택 후 `삭제`로 제거할 수 있습니다.

### 설정
- **일정 활성화**: 생성 직후 활성화 여부를 설정합니다.
- **일정 완료 후 작업**: `없음` 또는 `삭제`를 선택합니다.

> 💡 **참고:** `일정 완료 후 작업`은 `종료 일정`이 `종료 예약`인 경우에만 선택할 수 있습니다. `종료 없음`이면 해당 값은 `없음`으로 고정됩니다.

### 입력 필드 유효성
| 필드 | 허용 값                   | 검증 규칙 | 비고 |
| --- |------------------------| --- | --- |
| **이름** | 영문 대/소문자, 숫자, `-`, `_` | 최대 80자 | 필수 |
| **설명** | 문자열                    | 최대 200자 | 선택 |
| **일정 유형** | `CRON` \| `RATE`       | 둘 중 1개 선택 | 필수 |
| **CRON 표현식** | Cron 형식 문자열            | 유효한 Cron 표현식이어야 함 | `CRON` 선택 시 필수 |
| **RATE 값** | 정수                     | 1 이상 정수만 허용 | `RATE` 선택 시 필수 |
| **RATE 단위** | `분` \| `시간` \| `일`     | 단위 선택 필수 | `RATE` 선택 시 필수 |
| **시작 예약 일시** | 날짜/시간                  | `시작 일정`이 `시작 예약`이면 유효한 날짜/시간 필수 | 선택 |
| **종료 예약 일시** | 날짜/시간                  | `종료 일정`이 `종료 예약`이면 유효한 날짜/시간 필수 | 선택 |
| **종료 예약 일시** | 날짜/시간                  | 시작/종료 모두 예약이면 종료 > 시작 | 선택 |
| **서비스 선택** | 지원 서비스 4종              | 값 선택 필수 | 필수 |
| **일정 완료 후 작업** | `없음` \| `삭제`           | `종료 일정`이 `종료 예약`일 때만 변경 가능 | 선택 |

### 생성 버튼 활성화 조건
- 기본 정보(이름/설명) 유효성 통과
- 일정 패턴 유효성 통과
- 기간 유효성 통과
- 서비스 선택 완료

## 일정 편집

`일정 편집` 화면은 `일정 생성`과 동일한 UI를 사용하며, 기존 설정값을 초기값으로 표시합니다.

### 편집 버튼 활성화 조건
- `일정 생성`의 버튼 활성화 조건을 모두 만족
- 실제 변경사항이 1개 이상 존재

### 편집 모드 변경사항 판정 항목
- 이름, 설명
- 서비스
- 일정 유형/표현식
- 활성화 상태
- 일정 완료 후 작업
- 인스턴스 선택 목록
- 시작/종료 시각 (분 단위 기준)

## 일정 완료 시 동작
- 종료 시각이 지난 일정은 더 이상 실행되지 않습니다.
- `일정 완료 후 작업 = 없음`이면 마지막 실행 이후 일정이 자동 비활성화됩니다.
- `일정 완료 후 작업 = 삭제`이면 마지막 실행 이후 일정이 자동 삭제됩니다.

## 일정 활성화
목록에서 비활성화 상태 일정을 선택하고 `활성화`를 클릭합니다.

## 일정 비활성화
목록에서 활성화 상태 일정을 선택하고 `비활성화`를 클릭합니다.

## 일정 삭제
목록에서 삭제할 일정을 선택하고 `삭제`를 클릭한 뒤 확인 창에서 삭제를 확정합니다.

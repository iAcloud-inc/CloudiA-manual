# 외부 IP

`외부 IP` 메뉴는 현재 프로젝트에서 할당된 외부 IP를 조회하는 읽기 전용 화면입니다. 관리자 영역의 `외부 IP 풀`에서 준비된 외부 IP가 프로젝트에서 리소스들에 할당되면 이 메뉴에서 사용 현황을 확인할 수 있습니다.

## 화면 개요
- `목록 화면`만 제공됩니다.
- 실제 외부 IP 할당은 `플로팅 IP`, `포트 포워딩`, `로드 밸런서` 생성에서 이뤄집니다.

## 외부 IP 목록
상단에는 프로젝트 외부 IP 사용량 안내가 표시됩니다.

- **사용량 안내**: 프로젝트에 할당된 외부 IP 총량과 현재 사용 중인 수량을 표시합니다.

### 테이블 컬럼
| 컬럼           | 설명                                           |
|--------------|----------------------------------------------|
| **외부 IP 주소** | 프로젝트에서 할당된 외부 IP 주소입니다.                      |
| **ID**       | 외부 IP 고유 ID입니다.                              |
| **VPC**      | 현재 연결된 VPC입니다.                               |
| **사용 현황**    | `VPC`, `플로팅 IP`, `로드 밸런서`, `포트 포워딩` 중 하나입니다. |
| **연결된 리소스**  | 현재 이 외부 IP를 사용 중인 리소스의 이름과 ID 정보입니다.         |
| **생성 일시**    | 외부 IP가 할당된 날짜와 시각입니다.                        |

### 사용 현황 
| 사용 현황 | 의미                               |
| --- |----------------------------------|
| **VPC** | 외부 IP가 VPC에 점유되어 있는 상태입니다.       |
| **플로팅 IP** | 특정 플로팅 IP 리소스가 이 외부 IP를 사용 중입니다. |
| **로드 밸런서** | 외부 타입 로드 밸런서가 이 외부 IP를 사용 중입니다.  |
| **포트 포워딩** | 포트 포워딩 리소스가 이 외부 IP를 사용 중입니다.    |

## 참고 사항
- 외부 IP를 실제 서비스에 연결하려면 `플로팅 IP`, `포트 포워딩`, `로드 밸런서` 메뉴로 이동해야 합니다.
- 각 메뉴에서 외부 IP를 `자동 할당`으로 선택하면, 관리자 `외부 IP 풀`의 `시작 IP`가 작은 풀부터 검사하고 각 풀 내부도 시작 IP부터 순서대로 할당합니다.

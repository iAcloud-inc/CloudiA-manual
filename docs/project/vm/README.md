# 가상머신

프로젝트 `가상머신` 메뉴는 인스턴스 생성과 배치 정책, 템플릿, SSH 키, 오토 스케일링을 관리하는 핵심 운영 영역입니다. 대부분의 컴퓨트 작업이 이 섹션에서 시작됩니다.

## 구성 메뉴
- **인스턴스**: 가상머신 생성, 실행, 종료, 편집, 스냅샷 생성 등을 수행합니다.
- **시작 템플릿**: 반복 배포에 사용할 인스턴스 기본 구성을 저장합니다.
- **오토 스케일링 그룹**: 리소스 사용량에 따라 인스턴스 수를 자동 조정합니다.
- **선호도 그룹**: 인스턴스와 물리 머신의 배치 정책을 제어합니다.
- **SSH 키**: 인스턴스 접속에 사용할 공개키를 등록하고 관리합니다.

## 권장 작업 순서
1. 반복 배포가 많다면 `시작 템플릿`을 준비합니다.
2. 실제 컴퓨트 운영은 `인스턴스`에서 수행합니다.
3. 배치 규칙이 필요하면 `선호도 그룹`, 자동 확장이 필요하면 `오토 스케일링 그룹`을 사용합니다.

## 참고 사항
- 네트워크와 스토리지 메뉴를 함께 보며 운영하는 경우가 많으므로 종속 관계를 같이 확인하는 것이 좋습니다.

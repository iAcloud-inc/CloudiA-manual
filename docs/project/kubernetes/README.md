# 쿠버네티스

프로젝트 `쿠버네티스` 메뉴는 클러스터와 워커 노드, 스토리지 클래스를 포함한 쿠버네티스 운영 기능을 제공하는 영역입니다. 클러스터 생성부터 컴퓨트, 네트워크, 스토리지 관리까지 이 메뉴에서 수행합니다.

## 구성 메뉴
- **클러스터**: 쿠버네티스 클러스터 생성, 상세 조회, 워커 노드와 워커 오토 스케일링 그룹, 스토리지 자원을 관리합니다.

## 권장 작업 순서
1. VPC, 서브넷, 이미지, 계정 정보를 준비한 뒤 클러스터를 생성합니다.
2. 상세 화면에서 워커 노드와 네트워크, 스토리지 자원을 순차적으로 구성합니다.
3. 운영 중에는 모니터링, 컴퓨트, 리소스 탭을 함께 사용해 상태를 점검합니다.

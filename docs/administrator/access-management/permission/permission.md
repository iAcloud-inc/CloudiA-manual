# 권한

## 권한 목록

### 테이블 컬럼
| 컬럼 | 키 | 설명 |
|---|---|---|
| 메뉴 | `menu` | 클릭 시 권한 상세 화면으로 이동합니다. |
| 유형 | `authType` | `PROJECT` 또는 `ADMIN` 유형을 표시합니다. |
| 구분 | `perm` | 권한 세부 구분을 표시합니다. |
| 설명 | `description` | 권한 설명을 표시합니다. |
| 생성일 | `createdAt` | 생성 일시를 표시합니다. |

### 검색
- 검색 필드: `메뉴(menu)`, `구분(perm)`

## 권한 상세
- URL: `/administrator/access-management/permission/detail?id={permissionId}`
- 라우트 권한: `permission-view`

### 기본 정보
- 메뉴, 유형, 구분, 설명, 생성일

### 하단 연결 목록
- 권한 유형이 `ADMIN`이면 관리자 역할 목록을 표시합니다.
- 권한 유형이 `PROJECT`이면 역할 목록을 표시합니다.
- 목록 페이지 크기는 `5`입니다.
- 하단 목록은 조회 전용이며 이 화면에서 직접 편집하지 않습니다.

### 운영 기준
- 권한은 목록/상세 조회 전용으로 운영합니다.
- 실제 권한 부여/회수는 역할 또는 관리자 역할의 권한 편집에서 수행합니다.

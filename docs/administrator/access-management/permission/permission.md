# 권한

## 화면 개요

권한 화면에서는 관리자와 프로젝트 메뉴에 연결된 권한 항목을 조회하고 세부 권한 정보를 확인합니다.

## 권한 목록

### 검색
- 메뉴와 구분으로 권한을 검색할 수 있습니다.

### 테이블 컬럼
| 컬럼 | 설명 |
|---|---|
| 메뉴 | 클릭하면 권한 상세 화면으로 이동합니다. |
| 타입 | 관리자 권한인지, 프로젝트 권한인지 표시합니다. |
| 구분 | 조회, 편집, 삭제와 같은 권한 동작 구분을 표시합니다. |
| 설명 | 권한의 용도를 설명합니다. |
| 생성 일시 | 권한이 생성된 시각을 표시합니다. |

### 사용 시 참고
- 권한 화면은 조회 전용입니다.
- 권한 추가나 제거는 역할 또는 관리자 역할에서 진행합니다.

## 권한 상세

### 기본 정보
- 메뉴, 타입, 구분, 설명, 생성 일시를 확인할 수 있습니다.

### 연결된 역할
- 관리자 타입 권한이면 이 권한이 연결된 관리자 역할 목록을 표시합니다.
- 프로젝트 타입 권한이면 이 권한이 연결된 역할 목록을 표시합니다.

### 사용 시 참고
- 이 화면에서는 권한을 직접 수정하지 않습니다.
- 실제 권한 관리 작업은 역할 상세 또는 관리자 역할 상세의 `권한` 탭에서 진행합니다.

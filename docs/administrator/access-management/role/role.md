# 역할

## 역할 목록

### 테이블 컬럼
| 컬럼 | 설명 |
|---|---|
| 이름 | 클릭 시 역할 상세 화면으로 이동합니다. |
| 연결된 권한 수 | 역할에 연결된 권한 개수를 표시합니다. |
| 연결된 멤버 수 | 역할이 연결된 멤버 개수를 표시합니다. |
| 설명 | 역할 용도를 표시합니다. |
| 생성일 | 생성 일시를 표시합니다. |

### 검색
- 검색 필드: `이름(roleName)`

### 상단 액션 및 조건
| 액션 | 동작 | 활성 조건 |
|---|---|---|
| 생성 | 역할 생성 화면 이동 | 항상 활성 |
| 편집 | 역할 편집 화면 이동 | 1건 선택 + `system=false` |
| 삭제 | 삭제 확인 모달 오픈 | 1건 선택 + `system=false` |

### 제약
- 시스템 역할(`system=true`)은 편집/삭제할 수 없습니다.

## 역할 생성/편집

### 경로 및 권한
- 생성: `/administrator/access-management/role/create` (`role-edit`)
- 편집: `/administrator/access-management/role/update?id={roleId}` (`role-edit`)

### 입력 항목
| 항목 | 조건 |
|---|---|
| 이름* | 최대 20자 |
| 설명* | 최대 200자 |

### 생성 모드
- 권한 추가 섹션이 제공됩니다.
- 권한 선택 모달에서는 `PROJECT` 타입 권한만 선택할 수 있습니다.
- 선택한 권한은 로컬 리스트에서 삭제할 수 있습니다.

### 편집 모드
- 이름/설명만 수정할 수 있습니다.
- 시스템 역할 또는 조회 실패 케이스는 접근이 차단됩니다.

## 역할 상세

### 기본 정보
- 이름, ID, 연결 권한 수, 연결 멤버 수, 설명, 생성일, 수정일

### 하단 탭
- 권한 탭: 권한 리스트(페이지 크기 `5`) + 편집 버튼
- 멤버 탭: 연결 멤버 리스트(페이지 크기 `5`)

### 제약
- 시스템 역할은 권한 탭의 편집 버튼이 비활성화됩니다.

## 역할 권한 편집


### 주요 동작
- 현재 연결 권한을 로컬 리스트로 표시합니다.
- `추가` 버튼으로 권한 선택 모달을 열 수 있습니다.
- `삭제` 버튼으로 선택 권한을 제거합니다.
- `저장` 버튼으로 권한 ID 배열을 저장합니다.

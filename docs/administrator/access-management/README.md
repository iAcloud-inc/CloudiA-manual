# 접근 관리

관리자 `접근 관리` 메뉴는 멤버, 역할, 권한, 세션을 관리하는 영역입니다. 관리자 계정과 프로젝트 멤버의 접근 제어 정책을 운영할 때 사용하는 기본 메뉴입니다.

## 구성 메뉴
- **멤버**: 사용자 계정을 생성, 편집하고 상태를 관리합니다.
- **역할**: 프로젝트 멤버용 역할을 관리합니다.
- **관리자 역할**: 관리자 전용 역할을 관리합니다.
- **권한**: 역할에 연결할 세부 권한을 조회합니다.
- **세션**: 현재 활성 세션을 조회하고 강제 종료합니다.

## 권장 작업 순서
1. 필요한 권한 구성을 먼저 확인합니다.
2. 역할 또는 관리자 역할을 설계합니다.
3. 멤버에 역할을 연결합니다.
4. 운영 중 이상 접근은 세션 메뉴에서 확인합니다.

## 참고 사항
- `권한`은 조회 성격이 강하고, 실제 정책 구성은 `역할`, `관리자 역할`에서 수행합니다.
- 멤버 유형에 따라 상세 화면과 역할 연결 방식이 달라질 수 있습니다.

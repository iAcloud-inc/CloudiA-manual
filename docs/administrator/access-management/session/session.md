# 세션

## 세션 목록

### 테이블 컬럼
| 컬럼 | 설명 |
|---|---|
| 사용자명 | 세션을 시작한 사용자 계정명 |
| IP 주소 | 접속 클라이언트 IP |
| 세션 시작 시간 | 세션 생성 시각 |
| 마지막 활동 시간 | 마지막 활동 시각 |

### 검색
- 검색 필드: `사용자명(username)`, `IP 주소(ipAddress)`

### 상단 액션
| 액션 | 동작 | 활성 조건 |
|---|---|---|
| 세션 종료 | 선택 세션 종료 모달 오픈 | 1건 선택 |
| 새로고침 | 세션 목록 재조회 | 로딩 중 아님 + 클릭 후 1초 쿨다운 경과 |

### 세션 종료 모달
- 타이틀: `세션 종료`
- 본문에 선택한 사용자명을 포함한 로그아웃 확인 문구를 표시합니다.
- 확인 후 대상 세션을 즉시 종료합니다.

# 세션

## 화면 개요

세션 화면에서는 현재 로그인 세션을 조회하고 필요 시 사용자 세션을 종료할 수 있습니다.

## 세션 목록

### 검색
- 사용자 이름 또는 IP 주소로 세션을 검색할 수 있습니다.

### 주요 작업
- `세션 종료`: 선택한 세션을 종료합니다. 한 개를 선택했을 때만 사용할 수 있으며, 실행 전에 사용자 이름이 포함된 확인 문구가 표시됩니다.
- `새로고침`: 세션 목록을 다시 조회합니다. 조회 직후에는 잠시 비활성화됩니다.

### 테이블 컬럼
| 컬럼 | 설명 |
|---|---|
| 사용자 이름 | 세션을 시작한 사용자 이름을 표시합니다. |
| IP 주소 | 접속한 클라이언트 IP 주소를 표시합니다. |
| 세션 시작 시간 | 세션이 시작된 시각을 표시합니다. |
| 마지막 활동 시간 | 마지막 요청이 발생한 시각을 표시합니다. |

### 사용 시 참고
- 목록은 한 번에 한 개 세션만 선택할 수 있습니다.

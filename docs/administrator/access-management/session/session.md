# 세션

## 세션 목록
**세션 목록** 화면에서는 현재 CloudiA에 로그인된 사용자 세션들을 확인하고 관리할 수 있습니다.

세션 목록의 주요 컬럼은 다음과 같습니다:

| 컬럼            | 설명                                                           |
|---------------| -------------------------------------------------------------- |
| **사용자 이름**    | 세션을 시작한 사용자의 계정 이름입니다. |
| **IP 주소**     | 사용자가 CloudiA에 접속한 클라이언트 장치의 IP 주소입니다.                                |
| **세션 시작 시간**  | 해당 세션이 시작된 일시입니다.                       |
| **마지막 활동 시간** | 사용자가 마지막으로 활동한 시각입니다.        |

> 💡 **참고:** 세션 목록은 **단일 선택**만 지원합니다.
 
## 세션 종료 
- 선택한 세션을 강제로 종료합니다. 
- 세션은 한 번에 하나만 선택할 수 있으며 `세션종료`  버튼은 세션을 정확히 하나 선택했을 때만 활성화됩니다. 
- 버튼 클릭 시 세션종료 창이 나타나며, **`세션종료`**를 누르면 해당 사용자가 CloudiA에서 로그아웃됩니다.

# 관리자 역할

## 관리자 역할 목록

### 검색
- 이름으로 관리자 역할을 검색할 수 있습니다.

### 주요 작업
- `생성`: 새 관리자 역할을 등록합니다.
- `편집`: 선택한 관리자 역할의 이름과 설명을 수정합니다. 한 개를 선택했을 때만 사용할 수 있습니다.
- `삭제`: 선택한 관리자 역할을 삭제합니다. 한 개를 선택했을 때만 사용할 수 있습니다.

### 테이블 컬럼
| 컬럼 | 설명 |
|---|---|
| 이름 | 클릭하면 관리자 역할 상세 화면으로 이동합니다. |
| 연결 권한 수 | 이 관리자 역할에 연결된 권한 수를 표시합니다. |
| 연결 멤버 수 | 이 관리자 역할이 부여된 관리자 멤버 수를 표시합니다. |
| 설명 | 역할의 용도를 설명합니다. |
| 생성 일시 | 역할이 생성된 시각을 표시합니다. |

### 사용 시 참고
- 목록은 한 번에 한 개 역할만 선택할 수 있습니다.
- 시스템 관리자 역할은 Cloud:iA에서 기본 제공하는 관리자 역할입니다. 플랫폼 운영에 필요한 기본 관리자 권한이 포함되어 있으므로 편집하거나 삭제할 수 없습니다.

## 관리자 역할 생성/편집

### 입력 항목
- 기본 정보: 이름과 설명을 입력합니다.
- 생성 화면: 관리자 권한을 함께 추가할 수 있습니다.
- 편집 화면: 이름과 설명만 수정할 수 있습니다.

### 입력 규칙
- 이름: 최대 20자까지 입력할 수 있습니다.
- 설명: 최대 200자까지 입력할 수 있습니다.

### 권한 추가
- 생성 화면에서 `추가` 버튼을 눌러 권한 선택 창을 열 수 있습니다.
- 관리자 역할에는 관리자 권한만 추가할 수 있습니다.
- 선택한 권한은 목록에서 다시 삭제할 수 있습니다.

### 사용 시 참고
- 시스템 관리자 역할은 편집 화면으로 이동할 수 없습니다.
- 권한 변경은 관리자 역할 상세 화면의 `권한` 탭에서도 다시 수정할 수 있습니다.

## 관리자 역할 상세

### 기본 정보
- 이름, ID, 연결 권한 수, 연결 멤버 수, 설명, 생성 일시, 수정 일시를 확인할 수 있습니다.

### 하단 탭
- `권한`: 현재 연결된 관리자 권한 목록을 확인하고 `편집` 버튼으로 권한을 수정할 수 있습니다.
- `멤버`: 이 관리자 역할이 부여된 관리자 멤버 목록을 확인할 수 있습니다.

### 사용 시 참고
- 시스템 관리자 역할은 `권한` 탭의 `편집` 버튼이 비활성화됩니다.

## 관리자 역할 권한 편집

### 주요 작업
- 현재 연결된 관리자 권한 목록을 확인할 수 있습니다.
- `추가` 버튼으로 관리자 권한을 더 연결할 수 있습니다.
- `삭제` 버튼으로 선택한 권한을 목록에서 제거할 수 있습니다.
- `저장` 버튼을 눌러 변경 내용을 적용합니다.

### 사용 시 참고
- 관리자 역할에는 관리자 권한만 연결할 수 있습니다.
- 시스템 관리자 역할은 권한 편집 화면으로 이동할 수 없습니다.

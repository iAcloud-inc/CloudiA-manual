# 프로젝트

관리자 `프로젝트` 메뉴는 프로젝트 생성, 할당량 관리, 멤버 연결 상태를 운영하는 영역입니다. 관리자는 여기서 프로젝트 수명주기와 자원 상한을 직접 제어합니다.

## 구성 메뉴
- **프로젝트 관리**: 프로젝트 생성, 편집, 삭제와 할당량 조정, 멤버 연결을 관리합니다.

## 권장 작업 순서
1. 새 서비스나 조직 단위가 필요하면 `프로젝트 관리`에서 프로젝트를 생성합니다.
2. 프로젝트 운영 중에는 사용량 대비 할당량을 점검하고 필요 시 편집합니다.
3. 멤버와 역할 연결은 프로젝트 상세의 멤버 영역에서 함께 관리합니다.

## 참고 사항
- 관리자 영역의 프로젝트 관리는 생성과 할당량 제어가 중심입니다.
- 프로젝트 사용자 화면의 `프로젝트 관리`는 조회 성격이 강하므로 목적을 구분해 사용하는 것이 좋습니다.

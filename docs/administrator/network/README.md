# 네트워크

관리자 `네트워크` 메뉴는 Cloud:iA 전체 프로젝트가 공통으로 사용하는 외부 네트워크 자원과 관리자 단 보안 정책을 관리하는 영역입니다.

## 구성 메뉴
- **외부 IP 풀**: 프로젝트에 할당할 외부 IP 대역을 설정하고 사용 현황을 추적합니다.
- **방화벽**: 네트워크 노드에 적용되는 인바운드 허용/차단 규칙을 관리합니다.
- **보안그룹 템플릿**: 프로젝트 보안그룹 생성 시 가져올 수 있는 표준 규칙 묶음을 정의합니다.

## 운영 흐름
1. 외부 서비스 노출에 사용할 외부 IP 대역은 `외부 IP 풀`에서 먼저 준비합니다.
2. 관리자 영역 자체 보호 정책은 `방화벽`에서 관리합니다.
3. 프로젝트 운영팀이 반복적으로 사용할 인스턴스 보안 정책은 `보안그룹 템플릿`으로 표준화합니다.

## 참고 사항
- `외부 IP 풀`은 관리자만 생성/수정/삭제할 수 있으며, 프로젝트 화면에서는 할당된 외부 IP 현황만 조회합니다.
- `방화벽` 정책은 저장 즉시 반영되므로 운영 중인 서비스 영향도를 사전에 검토해야 합니다.
- `보안그룹 템플릿`을 수정해도 이미 생성된 프로젝트 보안그룹에는 자동으로 반영되지 않습니다.

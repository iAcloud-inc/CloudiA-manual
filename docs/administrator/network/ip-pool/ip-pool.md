# 외부 IP 풀
`외부 IP 풀`은 Cloud:iA가 프로젝트에 배정할 외부 IP 대역을 관리하는 메뉴입니다. 

## 외부 IP 풀 목록
### 주요 작업
- `생성`: 새 외부 IP 풀을 생성합니다.
- `편집`: 선택한 1개 풀의 설정을 수정합니다.
- `삭제`: 선택한 1개 풀을 삭제합니다. 사용 중인 외부 IP가 있으면 삭제할 수 없습니다.

### 테이블 컬럼
| 컬럼           | 설명                                                        |
|--------------|-----------------------------------------------------------|
| **IP 주소 범위** | 풀에서 실제 할당에 사용하는 시작 IP와 종료 IP를 `시작 IP ~ 종료 IP` 형식으로 표시합니다. |
| **ID**       | 외부 IP 풀의 고유 ID입니다.                                        |
| **IP/CIDR**  | 풀 전체가 속한 IP/CIDR 입니다.                                     |
| **기본 게이트웨이** | 풀과 연결된 게이트웨이 IP입니다.                                       |
| **주소 사용 현황** | 사용 중인 IP 수와 전체 풀 크기를 `사용 / 전체` 형식으로 표시합니다.                |
| **설명**       | 외부 IP 풀 설명입니다.                                            |
| **생성 일시**    | 외부 IP가 할당된 시각입니다.                                         |

## 외부 IP 풀 상세
### 상단 요약
- **IP 주소 범위**: 시작 IP와 종료 IP입니다.
- **ID**: 외부 IP 풀 ID입니다.
- **IP/CIDR**: 외부 IP 풀이 속한 전체 IP/CIDR입니다.
- **기본 게이트웨이**: 기본 게이트웨이 IP입니다.
- **주소 사용 현황**: 사용 중인 IP 수와 전체 IP 수입니다.
- **설명**: 외부 IP 풀 설명입니다.
- **생성 일시**: 외부 IP가 할당된 시각입니다.

### 하단 목록: 외부 IP
해당 풀에서 배정된 외부 IP를 확인할 수 있습니다.

| 컬럼 | 설명                                               |
| --- |--------------------------------------------------|
| **외부 IP 주소** | 실제 외부 IP 주소입니다.                                  |
| **ID** | 외부 IP 고유 ID입니다.                                  |
| **프로젝트** | 해당 외부 IP가 속한 프로젝트입니다.                            |
| **VPC** | 연결된 VPC입니다.                                      |
| **사용 현황** | `VPC`, `플로팅 IP`, `로드 밸런서`, `포트 포워딩` 중 하나로 표시됩니다. |
| **연결 리소스** | 현재 연결된 리소스의 이름과 ID 정보입니다.                        |
| **생성 일시** | 외부 IP가 할당된 시각입니다.                                        |

## 외부 IP 풀 생성
### 기본 정보
- **IP/CIDR**: 외부 IP 풀이 속할 전체 대역을 입력합니다. 예: `192.168.179.0/24`
- **IP 주소 범위**: 실제 할당에 사용할 시작 IP 주소와 종료 IP 주소를 입력합니다.
- **기본 게이트웨이**: IP/CIDR 대역 안에 있으면서 할당 범위 밖의 게이트웨이 IP를 입력합니다.
- **설명**: 외부 IP 풀 설명을 입력합니다. 선택 항목입니다.

### 입력 제약
- `IP 주소 범위`와 `기본 게이트웨이`는 `IP/CIDR`가 올바를 때만 입력할 수 있습니다.
- 시작 IP와 종료 IP는 IP/CIDR 범위 안에 있어야 하며, 종료 IP는 시작 IP보다 같거나 뒤에 있어야 합니다.
- 기본 게이트웨이는 IP/CIDR 범위 안에 있어야 하며, 시작 IP~종료 IP 범위와 겹치면 안 됩니다.
- 설명은 최대 200자까지 입력할 수 있습니다.

## 외부 IP 풀 편집
편집 화면에서는 다음 항목만 변경할 수 있습니다.

- **IP 주소 범위**: 시작 IP, 종료 IP
- **설명**

다음 항목은 편집할 수 없습니다.

- **IP/CIDR**
- **기본 게이트웨이**

## 외부 IP 풀 삭제
- 삭제는 한 번에 1개만 수행합니다.
- 선택한 풀에 사용 중인 외부 IP가 하나라도 있으면 삭제할 수 없습니다.
- 삭제 전에는 상세 화면 하단의 외부 IP 목록에서 사용 현황을 먼저 확인해야 합니다.

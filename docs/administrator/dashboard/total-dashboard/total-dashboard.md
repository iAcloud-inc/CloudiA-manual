## 모든 노드

모든 노드 패널에서는 클러스터에 등록된 전체 물리 노드 현황을 요약하여 보여줍니다. 
총 노드 수와 정상/비정상 노드 수를 한눈에 확인할 수 있으며, 패널 우측 상단의 [더보기] 버튼을 통해 노드 관리 페이지로 이동할 수 있습니다.

### 요약 정보

Cloud:iA에 등록된 **전체 물리 노드 현황**을 요약한 영역이며, [**더보기**] 클릭 시 **노드 관리 페이지**로 이동합니다.

### 테이블 컬럼
| 항목      | 설명                       |
| ------- | ------------------------ |
| **전체**  | 등록된 전체 물리 노드 수입니다.       |
| **정상**  | 현재 정상 상태의 노드 수입니다.       |
| **비정상** | 통신 불가 등 문제가 발생한 노드 수입니다. |

## 매니지먼트

### 요약 정보
**매니지먼트** 패널은 CloudiA 시스템을 운영하는 **매니지먼트 노드들의 상태 정보**를 제공합니다. 매니지먼트 노드의 호스트명과 현재 상태를 한눈에 확인할 수 있으며, 상태에 따라 색상 적용됩니다.

패널에는 이중화 환경 구성에 따라 **1개 또는 2개의  노드**가 카드 형태로 표시됩니다.

### 테이블 컬럼

| 컬럼 | 설명 |
|------|------|
| **호스트명** | 관리 노드의 이름입니다. 시스템 운영의 핵심 노드이며, 구성에 따라 하나 또는 두 개가 표시됩니다. |
| **상태** | 현재 운영 상태를 표시합니다. 정상일 경우 초록색 **“정상”**, 문제가 있을 경우 빨간색 **“비정상”**으로 표시됩니다. |

## CPU

### CPU 자원 요약

모든 노드 CPU 사용량을 시각화한 패널입니다.

### 사용률 상위 5

CPU 사용률이 높은 노드 순위를 보여주는 리스트입니다.

| 항목                  | 설명                                      |
| ------------------- |-----------------------------------------|
| **호스트명**            | 노드 이름이며 클릭 시 해당 노드 상세 화면으로 이동합니다.       |
| **사용률(%)**          | 노드 CPU 사용률이며, 수준별 색상(녹색/황색/적색)으로 표시됩니다. |
| **사용 중 코어 / 전체 코어** | 실제 사용 중 코어 수와 총 코어 수를 보여줍니다.            |

### 사용률 추이 그래프

* “**CPU 최근 사용률**” 그래프 제공
* 최근 시간 동안의 전체 CPU 사용률 변화 표시
* **80% 임계선** 포함
* 마우스 오버 시 시점별 상세 수치 확인 가능

## 메모리

### 메모리 자원 요약

CPU 패널과 동일한 구성으로 클러스터 전체 메모리 사용량을 제공합니다.

#### 사용률 상위 5

메모리 사용률 상위 5개 노드를 표시합니다.

| 항목                  | 설명                      |
| ------------------- |-------------------------|
| **호스트명**            | 노드 이름이며 클릭 시 해당 노드 상세 화면으로 이동합니다. |
| **사용률(%)**          | 메모리 사용률이며, 수준별 색상(녹색/황색/적색)으로 수준 표시됩니다. |
| **사용 메모리 / 전체 메모리** | 사용 중 용량 및 전체 메모리 용량입니다. |

#### 사용률 추이 그래프

* “**메모리 최근 사용률**” 그래프 제공
* 전체 메모리 사용률의 시간별 변화를 시각화
* 80% 기준선 포함
* 마우스 오버로 세부 수치 확인 가능

## 이벤트

### 최근 이벤트 목록

시스템에서 발생한 최근 **이벤트 최대 6개**가 표시됩니다.

| 항목         | 설명                        |
| ---------- | ------------------------- |
| **이벤트 이름** | 발생한 이벤트의 이름입니다. |
| **사용자**    | 이벤트를 발생시킨 사용자명입니다.      |
| **메시지**     | 이벤트 메시지 요약입니다.            |
| **발생 일시**     | 이벤트 발생 시각입니다.             |

* ERROR/CRITICAL 이벤트는 빨간색으로 강조
* 좌측 아이콘 색상으로 등급 표시
* [**더보기**] 선택 시 이벤트 전체 목록 페이지 이동

## 감사 로그

### 최근 감사 로그 목록

최근 **감사 로그 9건**이 표시되며, 사용자 활동 및 자원 조작 기록을 제공합니다.

| 항목             | 설명                          |
| -------------- | --------------------------- |
| **사용자**        | 작업을 수행한 사용자명입니다.             |
| **메시지** | 어떤 자원에 어떤 작업이 수행되었는지 표시합니다. |
| **상태(성공/실패)**  | 성공은 녹색, 실패는 붉은색으로 표시됩니다.    |
| **발생 일시**         | 작업 수행 시각입니다.                |

* 감사 로그 메시지 클릭 시 **감사 로그 상세 화면**으로 이동
* 상단의 [**더보기**] 선택 시 전체 목록 화면으로 이동

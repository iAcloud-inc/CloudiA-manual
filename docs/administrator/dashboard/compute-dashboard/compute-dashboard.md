# 컴퓨트 대시보드

**컴퓨트 대시보드**는 Cloud:iA의 컴퓨트 노드 상태와 인스턴스 분포를 한눈에 확인하는 화면입니다.
전체 노드 상태 요약, CPU/메모리 할당 현황, 노드별 인스턴스 상태를 통합해서 제공합니다.

---

## 컴퓨트 노드 상태

### 요약 정보

컴퓨트 노드 상태 패널에서는 등록된 컴퓨트 노드의 전체/정상/비정상 수를 확인할 수 있습니다.
패널 우측 상단의 [**더보기**] 버튼을 클릭하면 컴퓨트 노드 관리 페이지로 이동합니다.

### 테이블 컬럼

| 항목      | 설명                 |
|---------|--------------------|
| **전체**  | 등록된 컴퓨트 노드 수입니다.   |
| **정상**  | 현재 정상 상태의 노드 수입니다. |
| **비정상** | 정상 상태가 아닌 노드 수입니다. |

---

## CPU

### CPU 자원 요약

CPU 패널은 컴퓨트 노드의 할당 코어 대비 사용 코어 비율을 원형 차트로 제공합니다.

### 사용률 상위 5

CPU 할당 비율이 높은 상위 5개 컴퓨트 노드를 목록으로 제공합니다.
노드명을 클릭하면 해당 컴퓨트 노드 상세 화면으로 이동합니다.

| 항목                | 설명                      |
|-------------------|-------------------------|
| **노드명**           | 컴퓨트 노드 이름입니다.           |
| **할당률(%)**        | 할당 코어 대비 사용 코어 비율입니다.   |
| **할당 코어 / 전체 코어** | 사용 중인 코어 수와 전체 코어 수입니다. |

### 색상 기준

| 구간          | 의미 |
|-------------|----|
| **0~60%**   | 정상 |
| **61~80%**  | 주의 |
| **81~100%** | 위험 |

---

## 메모리

### 메모리 자원 요약

메모리 패널은 컴퓨트 노드의 할당 메모리 대비 사용 메모리 비율을 원형 차트로 제공합니다.

### 사용률 상위 5

메모리 할당 비율이 높은 상위 5개 컴퓨트 노드를 목록으로 제공합니다.
노드명을 클릭하면 해당 컴퓨트 노드 상세 화면으로 이동합니다.

| 항목                  | 설명                       |
|---------------------|--------------------------|
| **노드명**             | 컴퓨트 노드 이름입니다.            |
| **할당률(%)**          | 할당 메모리 대비 사용 메모리 비율입니다.  |
| **사용 메모리 / 전체 메모리** | 사용 중인 메모리와 전체 메모리 용량입니다. |

### 색상 기준

| 구간          | 의미 |
|-------------|----|
| **0~60%**   | 정상 |
| **61~80%**  | 주의 |
| **81~100%** | 위험 |

---

## 노드별 인스턴스 현황

### 요약 정보

노드별 인스턴스 현황 패널에서는 각 컴퓨트 노드에 배치된 인스턴스를 육각 타일 형태로 표시합니다.
노드명을 클릭하면 해당 컴퓨트 노드 상세 화면의 인스턴스 탭으로 이동할 수 있습니다.

### 상태 범례

| 범례       | 포함 상태                                           |
|----------|-------------------------------------------------|
| **실행 중** | 실행 중                                            |
| **주의**   | 부팅 중, 종료 중, 생성 중, 삭제 중, 재부팅 중, 업데이트 중, 마이그레이션 중 |
| **에러**   | 에러                                              |
| **정지**   | 종료됨, 일시 중지, 정지됨                                 |

### 참고 사항

- 정상 상태의 컴퓨트 노드만 노드별 인스턴스 현황 패널에 표시됩니다.
- 노드당 최대 21개 인스턴스 타일을 표시합니다.
- 인스턴스가 많을 경우 마지막 타일에 `···`가 표시되며 클릭 시 상세 화면으로 이동합니다.
- 보안 인스턴스는 방패 아이콘으로 표시됩니다.

## 자동 갱신

### 요약 정보

- 기본 자동 갱신 주기는 5초입니다.
- 자동 갱신 주기는 **꺼짐, 5초, 10초, 30초, 1분, 5분, 15분, 30분, 1시간** 중에서 선택할 수 있습니다.
- 새로고침 버튼 클릭 시 즉시 최신 데이터를 다시 조회합니다.
- 마지막 갱신 시각을 함께 제공합니다.

# 대시보드

관리자 `대시보드` 메뉴는 Cloud:iA 전체 인프라의 상태를 한눈에 파악하는 요약 화면입니다. 노드 수, 자원 사용률, 최근 이벤트와 감사 로그를 빠르게 확인할 때 사용합니다.

## 구성 메뉴
- **전체**: 전체 물리 노드와 매니지먼트 노드, 자원 사용률, 최근 이벤트와 감사 로그를 통합해 보여줍니다.
- **컴퓨트 대시보드**: 컴퓨트 노드 상태와 CPU/메모리 사용률, 노드별 인스턴스 현황을 보여줍니다.

## 참고 사항
- 대시보드는 운영 현황을 요약해 보여주는 조회 전용 화면입니다.
- 상세 원인 분석이 필요하면 `이벤트`, `감사 로그`, `컴퓨트 노드`, `노드` 메뉴와 함께 확인하는 것이 좋습니다.

# 스토리지

관리자 `스토리지` 메뉴는 공통 스토리지 자원과 배포용 이미지 자산을 관리하는 영역입니다. 스토리지 도메인 준비와 ISO, 공용 이미지 배포를 담당합니다.

## 구성 메뉴
- **스토리지 도메인**: 이미지, 블록, 파일 시스템이 실제로 저장될 스토리지 도메인을 등록합니다.
- **공용 이미지 관리**: 모든 프로젝트에서 공통으로 사용할 OS 이미지를 관리합니다.
- **ISO**: ISO 이미지 파일을 등록하고 삭제합니다.

## 권장 작업 순서
1. 먼저 `스토리지 도메인`을 준비합니다.
2. 운영에 필요한 ISO와 공용 이미지를 해당 도메인에 등록합니다.
3. 프로젝트 사용자에게 공통 제공할 이미지는 공용 이미지로 관리합니다.

## 참고 사항
- 스토리지 도메인은 다른 스토리지 리소스의 기반이므로 삭제 전에 연관 리소스를 반드시 확인해야 합니다.
- ISO와 공용 이미지는 프로젝트 운영에 직접 노출되므로 파일 품질과 저장 위치를 함께 관리하는 것이 좋습니다.

# ISO

**ISO** 메뉴는 관리자 영역에서 ISO 이미지를 등록하고 삭제하는 기능을 제공합니다.  
등록된 ISO는 가상머신 설치/구성 시 사용할 수 있습니다.

## ISO 목록
### 주요 작업
- `생성`: 새로운 ISO를 등록합니다.
- `삭제`: 선택한 ISO를 삭제합니다.

> 💡 **참고:** 목록은 단일 선택으로 동작하며, 항목 선택 시 `삭제` 버튼이 활성화됩니다.

### 검색
- **검색 조건**: `이름`
- **검색어 입력** 후 `검색`을 실행합니다.

### 테이블 컬럼
| 컬럼 | 설명 |
| --- | --- |
| **이름** | ISO 이름입니다. |
| **ID** | ISO 고유 ID입니다. |
| **OS 타입** | ISO의 OS 타입(Variant)입니다. |
| **크기** | ISO 파일 크기입니다. |
| **스토리지 도메인** | ISO가 저장된 스토리지 도메인입니다. |
| **생성 일시** | ISO 등록 시각입니다. |

## ISO 생성
### 기본 정보
- **스토리지 도메인**: ISO를 저장할 스토리지 도메인을 선택합니다.
- **OS 타입**: 제공되는 OS 타입 목록에서 선택합니다.
- **ISO 파일**: `.iso` 파일을 업로드합니다.

### 입력 필드 유효성
| 필드 | 허용 값 | 검증 규칙 | 비고 |
| --- | --- | --- | --- |
| **스토리지 도메인** | 등록된 스토리지 도메인 | 1개 선택 필수 | 필수 |
| **OS 타입** | 시스템 제공 OS 타입 | 1개 선택 필수 | 필수 |
| **ISO 파일** | `.iso` 확장자 파일 | 파일 선택 필수 | 필수 |

> ⚠️ **주의:** ISO 생성 시 `CEPH` 타입 스토리지 도메인은 선택할 수 없습니다.

### 생성 버튼 활성화 조건
- 스토리지 도메인, OS 타입, ISO 파일을 모두 입력하면 `생성` 버튼이 활성화됩니다.

## ISO 삭제
1. 목록에서 삭제할 ISO를 선택합니다.
2. `삭제`를 클릭합니다.
3. 삭제 확인 창에서 내용을 확인한 뒤 삭제를 진행합니다.

> ⚠️ **주의:** 선택한 ISO의 삭제 가능 조건을 만족하지 않으면 확인 창의 삭제 버튼이 비활성화됩니다.

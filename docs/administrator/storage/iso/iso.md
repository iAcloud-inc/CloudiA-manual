# ISO

`ISO`는 관리자 영역에서 가상머신 설치에 사용할 ISO 이미지를 등록하고 관리하는 화면입니다. 등록된 ISO는 프로젝트에서 ISO 설치 유형 인스턴스를 만들 때 사용할 수 있습니다.

## 화면 개요
- `목록 화면`: ISO 현황을 조회하고 생성, 삭제 작업을 시작합니다.
- `생성 화면`: 스토리지 도메인, OS 타입, ISO 파일을 지정해 새 ISO를 등록합니다.

## ISO 목록
### 주요 작업
- `생성`: 새로운 ISO를 등록합니다.
- `삭제`: 선택한 ISO를 삭제합니다.

> 💡 **참고:** 목록은 단일 선택으로 동작하며, 항목 선택 시 `삭제` 버튼이 활성화됩니다.

### 검색
- **검색 조건**: `이름`
- **검색어 입력**: 검색어를 입력합니다.
- **검색 버튼**: 조건에 맞는 ISO를 조회합니다.

### 테이블 컬럼
| 컬럼 | 설명 |
| --- | --- |
| **이름** | ISO 이름입니다.<br>생성 시 사용한 ISO 파일 이름과 동일합니다. |
| **ID** | ISO 고유 ID입니다. |
| **OS 타입** | ISO 등록 시 선택한 OS 타입(Variant)입니다. |
| **크기** | ISO 파일 크기입니다. |
| **스토리지 도메인** | ISO가 저장된 스토리지 도메인입니다. |
| **생성 일시** | ISO를 등록한 일시입니다. |

## ISO 생성
### 기본 정보
- **스토리지 도메인**: ISO를 저장할 스토리지 도메인을 선택합니다.
- **OS 타입**: 제공되는 OS 타입을 목록에서 선택합니다.
- **ISO 파일**: `.iso` 파일을 업로드합니다.

### 입력 필드 유효성
| 필드 | 허용 값 | 검증 규칙 | 비고 |
| --- | --- | --- | --- |
| **스토리지 도메인** | 선택 가능한 스토리지 도메인 | 필수입니다. 1개를 선택합니다. | `CEPH`는 지원하지 않습니다. |
| **OS 타입** | 시스템 제공 OS 타입 | 필수입니다. 1개를 선택합니다. | ISO 분류에 사용합니다. |
| **ISO 파일** | `.iso` 파일 | 필수입니다. 파일을 선택해야 합니다. | 업로드 대상입니다. |

> ⚠️ **주의:** ISO 생성 시 `CEPH` 타입 스토리지 도메인은 선택할 수 없습니다.

### 생성 시 참고 사항
- 모든 필수 항목을 입력하면 `생성` 버튼이 활성화됩니다.
- 등록된 ISO는 프로젝트 인스턴스 생성 화면에서 ISO 설치 유형에 사용할 수 있습니다.


## ISO 삭제
1. 목록에서 삭제할 ISO를 선택합니다.
2. `삭제`를 클릭합니다.
3. 삭제 확인 창에서 내용을 확인한 뒤 삭제를 진행합니다.

> ⚠️ **주의:** 연결된 인스턴스가 없는 ISO만 삭제할 수 있습니다.

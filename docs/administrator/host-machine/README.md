# 물리 머신

관리자 `물리 머신` 메뉴는 Cloud:iA 인프라를 구성하는 실제 호스트와 컴퓨트 노드, 노드 그룹을 관리하는 영역입니다. 물리 자원 상태 확인, 전원/BMC 관리, 노드 그룹 운영에 사용합니다.

## 구성 메뉴
- **노드**: 전체 물리 머신의 전원, BMC, 하드웨어 상태를 조회하고 관리합니다.
- **노드 그룹**: 노드를 논리적으로 묶어 운영 단위로 관리합니다.
- **컴퓨트 노드**: 가상머신이 실제로 배치되는 컴퓨트 노드의 상태와 설정을 관리합니다.

## 참고 사항
- `노드`와 `컴퓨트 노드`는 대상이 겹치지만, 노출하는 관리 기능이 다릅니다.
- 전원, BMC, 마이그레이션 같은 작업은 실제 서비스 영향이 있으므로 수행 전에 상태를 반드시 확인해야 합니다.
